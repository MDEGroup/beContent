HTML Structure

This theme is a fixed layout with two columns (3 in homepage). In subpage, all of the information within the main content area (column #2) is nested within a div with an id of "right". The sidebar's (column #1) content is within a div with an id of "left".

Homepage layout structure

Subpage layout structure

If you would like to edit the color, font, or style of any elements in one of these columns, open default.css in /css folder and do the following: If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.
So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

Javascript

This theme imports four Javascript files in "/js" folder.

1) jQuery file
2) 2 prettyPhoto js files
3) Easy Slider jquery plugin

1) jQuery is a Javascript library that greatly reduces the amount of code that you must write. Visit http://jquery.com for more informations.

2) In portfolio page ("portfolio.htm") I'm using a prettyPhoto script, which allow a nice image resizing effect. The code needed for run the script is nested within and in document head. PSD Files

I've included some psds with this theme in "/psds" folder:

1) The homepage layout (LIGHT_homepage.psd)
2) The subpage layout (LIGHT_subpage.psd)
3) The homepage layout in dark version (DARK_subpage.psd)
4) The subpage layout in dark version (DARK_subpage.psd)
5) And some another psds in psds/images folder

If you'd like to make some changes in layout, open "homepage.psd" or "subpage.psd", make the necessary adjustments and using slice tool or crop tool (recomended) slice the layout to png images.
